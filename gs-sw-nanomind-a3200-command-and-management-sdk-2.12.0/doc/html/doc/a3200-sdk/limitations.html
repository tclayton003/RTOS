1.3. Known Limitations¶

1.3.1. AVR32 toolchain¶

The toolchain has been upgraded to a GomSpace patched version (3.4.2_gs1), which solves the following issues when running in a multi-threaded environment;

  • Enabling FreeRETOS configUSE_NEWLIB_REENTRANT causes a crash when a task/thread terminates. This is due to a bug in newlib 1.16 (part of the toolchain). configUSE_NEWLIB_REENTRANT allows formatting of float/double from multiple threads, without crashing.
  • Missing file locking, when accessing files from mulitple threads. The patched toolchain now supports retargetable locks in newlib, and the locks has been implemented in libembed.

The new toolchain 3.4.2_gs1 comes with NanoMind A3200 BSP (a3200-bsp) version 2.5.1. The toolchain source files are available on https://github.com/GomSpace/avr32-toolchain.

1.3.2. FreeRTOS¶

FreeRTOS has been upgraded to version 10.2, to solve some stability issues with 8.0. The FreeRTOS version can be changed through the wscript:

# Options for libasf
ctx.options.freertos_version = 10

1.3.2.1. FreeRTOS runtime stats¶

Runtime stats (Time % from the ps command) is disabled (in liba3200/conf/FreeRTOSConfig.h) for the time being.
