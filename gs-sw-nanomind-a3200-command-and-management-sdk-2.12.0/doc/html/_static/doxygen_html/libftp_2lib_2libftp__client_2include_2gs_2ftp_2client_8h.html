enum gs_ftp_info_type_t

Indicates the type of info provided in gs_ftp_info_callback.

Enumerator
GS_FTP_INFO_FILE 

File size and checksum.

GS_FTP_INFO_CRC 

Checksum of remote and local file.

GS_FTP_INFO_COMPLETED 

Completed and total chunks.

GS_FTP_INFO_PROGRESS 

Current chunk, total_chunks and chunk_size.

enum gs_ftp_mode_t

FTP mode.

The FTP mode is solely used for compensating for none standard FTP settings and behaviors, e.g. legacy quirks.

Enumerator
GS_FTP_MODE_STANDARD 

Standard FTP mode.

GS_FTP_MODE_GATOSS 

Legacy GATOSS mode.

GATOSS uses a different FTP port, and different backend-type for files.
