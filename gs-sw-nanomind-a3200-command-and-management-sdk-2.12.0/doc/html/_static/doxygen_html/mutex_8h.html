mutex.h File Reference

Detailed Description

Mutex (recursive).

The mutex API wraps POSIX pthread_mutex and FreeRTOS mutex.

Note
Mutex can not be used from within an ISR routine - use gs_sem instead.
#include <gs/util/error.h>
#include <pthread.h>
