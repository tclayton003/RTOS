Function Documentation

void gs_a3200_adc_channels_init ( void  )

Initialise ADC.

This function must be called before any other function in this file. This will call adc_channels_calibrate, so power switches must be OFF. Also, the adc_channels_ functions should have exclusive access to the ADC hardware, or it will not work.

gs_error_t gs_a3200_adc_channels_sample ( int16_t *  adc_values)

Get a single sample from each channel.

This function does a blocking sample on all used adc channels and writes those to the array pointed to by adc_values. This function must only be called from Task context.

Parameters
[out]adc_valuesPointer to array of dimension GS_A3200_ADC_NCHANS
