#define CSP_BUFFER_PACKET_OVERHEAD   (sizeof(csp_packet_t) - sizeof(((csp_packet_t *)0)->data))

Size of the packet overhead in csp_packet_t.

The overhead is the difference between the total buffer size (returned by csp_buffer_size()) and the data part of the csp_packet_t (returned by csp_buffer_data_size()).
