#define GS_PARAM_STORE_F_SETUP_AUTO_PERSIST   0x2000

Configure auto-persist store and callback.

This is set during gs_param_table_create(), when check is performned on any configured store(s).

#define GS_PARAM_STORE_F_SKIP_AUTO_PERSIST   0x1000

Store flags.

Flags must be in range: bit 24 - 31, to avoid clash with other parts of the parameter system. Skip loading auto-persist parameters.
