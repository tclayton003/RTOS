enum gs_autodeploy_state_machine_status_t

Generic auto deploy status codes.

Enumerator
FINISHED_WITH_I2C_ERROR 

Finished deploy algorithm with an I2C error.

FINISHED_UNSUCCESSFULLY 

Finished deploy algorithm some undeployed items.

FINISHED_SUCCESSFULLY 

Succesfull deployment of all items.

WAITING 

Deploy algorithm waiting to start.

DEPLOYING 

Deployment is ongoing.

Auto deploy two DSPs (deployable solar panels), which relates to four Interstage 3u (AR6) boards.

Note
This function start deploying immediately and takes as long time as the algorithm takes to finish.
Parameters

i2c_addr_12I2C address for second board of first DSP.
