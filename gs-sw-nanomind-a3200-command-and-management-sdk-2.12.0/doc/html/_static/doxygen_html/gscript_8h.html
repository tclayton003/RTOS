Detailed Description

gscript (GOMspace script).

Gscripts consists of a number of GOSH commands and some simple control logic. The format is as follows: <delay ms> <retry> <goto on error> <comamnd> [options] delays ms: delay in mS before the command is executed. retry: number of times the command is tried executed, if it fails. goto on error: which line to goto, if command fails (after retry). command: GOSH command options: option(s) for command.
