bool gs_csp_service_dispatcher_conf_t::bind_any

Bind to any unbound ports (CSP_ANY).

Only one task/dispatcher can bind to any ports. This can be used to log unexpected incoming connections.

unsigned int(* gs_csp_service_dispatcher_conf_t::callback) (void)

Callback after timeout or processsed connection.

The return value is the next timeout in milli seconds. Return UINT32_MAX to use default timeout.

bool gs_csp_service_dispatcher_conf_t::disable_watchdog

Disable watchdog.

The watchdog is created with a fixed timeout of 20 seconds.

size_t gs_csp_service_dispatcher_conf_t::listen_backlog

CSP connection backlog.

If 0 (zero), the backlog is set to 10.
