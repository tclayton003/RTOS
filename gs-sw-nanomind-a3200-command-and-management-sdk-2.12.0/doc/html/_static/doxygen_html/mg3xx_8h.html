Enumeration Type Documentation

enum gs_mg3xx_filter

Digital filter.

This filter type both controls the digital low pass filter and down sampling, as these must have a valid combination.

Enumerator
NO_FILTER_1000SPS 

No filter and down sampled to 1000 samples per second.

KAISER_FC50_TAP_32_125SPS 

FIR kaiser filter with 32 taps and cutoff frequency at 50 hz and down sampled to 125 samples per second.
