adc.h File Reference

Detailed Description

STM32L4 ADC API.

This API includes initialization routine of ADC and both raw sample functions together with voltage conversion and averaging functions.

#include <stm32l4xx_hal.h>
#include <gs/util/drivers/gpio/gpio.h>

typedef enum gs_stm32l4_adc_sample_time gs_stm32l4_adc_sample_time_t

ADC sample time.

Number of prescaled ADC clock cycles for multiplexing to right source and charging internal capacitor before sampling. Can be set for each channel.
