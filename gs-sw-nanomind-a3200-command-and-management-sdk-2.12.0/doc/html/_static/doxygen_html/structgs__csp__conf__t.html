size_t gs_csp_conf_t::csp_buffer_size

Size of a CSP buffer's data part (in bytes).

Corresponds to data_size in the csp_buffer_init().

size_t gs_csp_conf_t::csp_buffers

Number of CSP buffers to allocate.

Corresponds to buf_count in csp_buffer_init().

uint32_t gs_csp_conf_t::default_connection_options

Default (minimum) connection options.

Options will always be or'ed onto new connections, see csp_connect(). Maps to conn_dfl_so in csp_conf_t.

const char* gs_csp_conf_t::hostname

Host name, returned by the CSP_CMP_IDENT request.

Note
String must remain valid as long as the application is running.
uint8_t gs_csp_conf_t::max_connections

Max number of connections.

A fixed connection array is allocated by csp_init() Maps to conn_max in csp_conf_t.

uint8_t gs_csp_conf_t::max_rdp_window_size

Max RDP window size.

Maps to rdp_max_window in csp_conf_t.

const char* gs_csp_conf_t::model

Model, returned by the CSP_CMP_IDENT request.

Note
String must remain valid as long as the application is running.
const char* gs_csp_conf_t::revision

Revision, returned by the CSP_CMP_IDENT request.

Note
String must remain valid as long as the application is running.
uint8_t gs_csp_conf_t::router_queue_length

Length of routers incoming message queue (max queue messages).

Will be adjusted, if bigger than csp_buffers Maps to fifo_length in csp_conf_t.

uint8_t gs_csp_conf_t::rx_queue_length

Rx queue length (max queued messages).

Will be adjusted, if bigger than csp_buffers Maps to conn_queue_length in csp_conf_t.

bool gs_csp_conf_t::use_command_line_options

Use command line options.

Configure interfaces specified on the command line.

Note
Only supported on Linux - ignored on platforms without command line options.
bool gs_csp_conf_t::use_gs_log

Forward CSP logs to GomSpace log (libutil).

Deprecated:
version 3.0, ignored - as libcsp is compiled with GomSpace log macros (defining CSP_USE_EXTERNAL_DEBUG).
