gs_param_callback_func_t gs_param_table_create_t::callback

Parameter update callback.

Called when a parameter is set (even if the value is the same). The callback will also be called after load of the table, unless suppresssed by GS_PARAM_F_NO_CALLBACK.

gs_param_table_initializer_function_t gs_param_table_create_t::initializer

Function for initializing table parameters.

Note
If using gs_param_set(), use the flag GS_PARAM_F_NO_CALLBACK to avoid callbacks. Unless the table memory is allocated (by GS_PARAM_TABLE_F_ALLOC_MEMORY) or initialized using default_values, the table memory hasn't been touched and the initializer must set all values.
