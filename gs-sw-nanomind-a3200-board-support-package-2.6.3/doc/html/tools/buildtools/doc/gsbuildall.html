4.1.5. Setting up gsbuildall.py¶

This sections contains documentation for various commands useful when creating a gsbuildall.py file.

gs.buildtools.util.waf_command(options=[], cmd=['distclean', 'configure', 'clean', 'build'], dir='.', recurse=False, force_docker_img=None)¶

Run a waf command

Parameters:
  • options – a list of command line options passed to waf
  • cmd – a list of subcommands to run with waf, useful when wanting to build doc or gs_dist
  • dir – the working directory for the waf command.
  • recurse – If True run the waf command in all subfolder with a wscript
  • force_docker_img – If set the waf command will be executed using the specified docker image
