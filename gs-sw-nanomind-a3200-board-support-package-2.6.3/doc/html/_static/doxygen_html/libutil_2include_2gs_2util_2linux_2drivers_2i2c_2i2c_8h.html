gs_error_t gs_i2c_master_register_driver ( const gs_i2c_master_driver_entry_t *  driver_entry)

Register a master driver.

A specific driver can be assigned to a specific address and device or it can be registered to every address on a device or every address on every device.

The latest registered driver, which fit the device an address, is the one used.

Parameters
[in]driver_entrydriver and configuration to be registered
Returns
GS_OK on success
other error code on failure
