spn_fl512s.h File Reference

Detailed Description

Interface for S70FL01GSAGMFI011 (1 Gbit (128 MB) 3.0V SPI Flash).

The S70FL01S consists of 2 FL512S dies with individual chip selects. Each die is 512 Mbit (64 MB) divided into 256 sectors (or blocks) of 256 KB. A sector is the smallest erase unit. Each sector is split into 512 pages of 512 bytes.
