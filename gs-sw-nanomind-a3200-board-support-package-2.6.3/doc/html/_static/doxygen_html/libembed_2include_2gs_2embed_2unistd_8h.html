gs_error_t gs_setcwd ( const char *  cwd)

Set current working directory.

This API can be used on embeeded platforms with a limited file-system, where standard getcwd() isn't supported. The cwd will be stored in a blobal variable and returned by gs_getcwd().

Note
Not thread-safe.
Parameters
[in]cwdcurrent working directory.
Returns
GS_ERROR_NOT_FOUND if no current directory is set.
GS_OK on success
other error code on failure
