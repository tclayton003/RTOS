service_dispatcher.h File Reference

Detailed Description

CSP Service Dispatcher.

The dispatcher is a task/thread, listening on a configured number of ports and forwards the the incoming connection to a configured CSP service handler. The dispatcher touches a software watchdog for every handled connection.
