gs_reset_cause_store_t Struct Reference

Detailed Description

Generic GomSpace structure to hold reset cause during SW reset.

This structure is used on platforms that does not have scratchpad registers to store the information. Instead it is kept in a special internal RAM section which is not overwritten during SW reset.
